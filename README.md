**The Patakyno Game — точка невозврата**

Интерактивное приложение-"игра" с юмористическим подтекстом. После того, как пользователь соглашается стать "Патакой", сценарий выполняет ряд системных команд для пугающего, но безвредного розыгрыша.

**⚠️ Предупреждение: это НЕ настоящая игра!**

Приложение предназначено исключительно для образовательных и развлекательных целей. Вы запускаете его на свой страх и риск.

**Функционал:**

При нажатии на кнопку "Да" в сплывающем окне с вопросом "Стать Патакой? произойдут следующие действия;
1. Всплывающие окна;
2. Смена обоев пользователя;
3. Проигрывание звука;
4. Модификация акцентного цвета системы в реестре;
5. Перезагрузка системы.

**Создано на Python.**
