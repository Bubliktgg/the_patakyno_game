# English:

## The Patakyno Game — point of no return

A Python-based interactive "game" with a humorous twist. Choosing "YES" is an irreversible decision that demonstrates your unwavering dedication to the cause of Patakism XD.

## ⚠️ Disclaimer: this is NOT a real game!

The app is intended solely for educational and entertainment purposes. Use at your own risk.

[Download page](https://github.com/Bubliktgg/the_patakyno_game/releases/tag/v4.0)

## Features:

Clicking "Yes" in the pop-up window with the question "Become a Pataka?" the following actions will occur:
1. Pop-up windows;
2. Desktop wallpaper modification;
3. Audio playback;
4. Modification of the system's accent color in the registry;
5. System restart.

Created in Python.
